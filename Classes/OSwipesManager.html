Overview

The Operators Swipes Manager is an abstraction of swipe operations (in 4 directions). It will allow the swiping away of the currently viewed card (or View), to the top, bottom, left or right of the screen (but only if the message [OSwipesManagerDelegate isAllowed:Directions] returns true).
